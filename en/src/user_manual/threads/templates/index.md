# Discussion templates

If you frequently use a particular discussion, workflow or decision making process, you can now save this as a discussion template.  

Build and save your discussion template with the content and polls appropriate to your process.

*On this page*
- [Set up a discussion template](#set-up-a-discussion-template)
- [Using discussion templates](#using-discussion-templates)

## Set up a discussion template

To set up your own template, start a discussion in your group as usual.  

Go to the **Settings** tab and check the **Template** box.

![](template_start.png#width-80)

Back on the **Content** tab, add a category tag(s), title and provide some sample text to help people using the template.

![](template_start_2.png#width-80)

Click **Start discussion** and your new discussion template is created.

The discussion is clearly marked as a template.  Note the statement at the top of the discussion page and the **Template** tag alongside the discussion privacy tag.

![](template_thread_outline.png#width-80)

There is value to easily copy and share a discussion name, tags and context that is frequently used.

However, the real benefit comes when you add the polls for a particular decision or workflow process, so they are readily available for people using the template.  Add the polls in the order of your process.

In the video example of the [Simple decision process](https://help.loomio.com/en/user_manual/polls/decisions/index.html), the first poll is a Sense check:

![](template_sense_check.png#width-80)

Note that **Voting open** is unticked - this is a template discussion, so not used for actual voting.

**Save poll**.

As voting is not open, you can invite people to provide feedback on the poll if you wish.

Repeat by adding further polls as needed.

The discussion template will then contain the content and polls typically used in your decision or workflow process.

![](template_thread_ready.png)

## Using discussion templates

When a discussion template has been created in your group, anyone in your group starting a new discussion will see a new screen, asking if they want to start the discussion as either:
- Blank - discussion starts as usual with no pre-filled content
- Simple decision process - your new discussion template
- New template - to start a new template

![](template_popup.png#width-80)

When a discussion template is selected, a copy of the discussion template content is created.

Edit discussion title and context as usual.

![](template_thread_copy.png#width-80)

The polls created in the template are now ready to use, in the order they were created.

![](template_polls.png#width-80)

You can see an example of this discussion template in use in the [demo group](https://www.loomio.com/demo).  See the discussion "Refreshing our brand".

### Close discussion template

As a last step, **Close** the discussion.  The discussion template is accessible with **New discussion** as above, but is removed from the open discussions on your group page.

Select **closed** to view the template.

![](template_closed.png#width-80)
